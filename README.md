# AdidasBot
Browser automation script for buying adidas shoes and checking restocks

## Prerequisites
selenium module
```
pip install selenium
```

webdriver_manager module
```
pip install webdriver_manager
```

## Setup
1) Clone respository
```
git clone https://github.com/gdotzheng/AdidasBot.git
```
2) Edit ```details.py```

3) Run the script
```
python adidasbot.py
```
